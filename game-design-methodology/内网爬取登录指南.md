# 内网爬取登录指南（看不到浏览器时必读）

云 Agent / 远程 Linux **没有可见桌面**，弹出的浏览器你看不到，无法在云端完成 iOA 扫码。

**正确做法：在本机登录一次，导出 Cookie，再让云环境用 Cookie 爬取。**

---

## 方案 A：本机登录 + 云环境爬取（推荐）

### 第 1 步：在本机导出登录状态

需要：公司电脑、已安装 **iOA**、能打开 `https://game.woa.com/knowledge/1013`

```bash
# 克隆或进入项目目录
cd game-design-methodology

pip install playwright
playwright install chromium

python3 scripts/save_woa_login.py
```

1. 会弹出**本机**浏览器
2. 完成 iOA 登录，确认知识库页面正常显示
3. 终端按 **Enter**
4. 生成 `game-design-methodology/woa_auth.json`

### 第 2 步：上传 `woa_auth.json`

把 `woa_auth.json` 上传到云工作区的 `game-design-methodology/` 目录（拖拽到 Cursor 文件树即可）。

> 该文件含登录 Cookie，**不要提交到 Git**（已在 .gitignore 中）。

### 第 3 步：在云环境运行爬虫

```bash
cd game-design-methodology
python3 scripts/crawl_woa_knowledge.py \
  --auth-file woa_auth.json \
  --headless \
  --start-url https://game.woa.com/knowledge/1013
```

结果输出到 `raw-articles-woa/`。

---

## 方案 B：全程在本机爬取

若本机已连 iOA，可直接在本机跑完整流程，无需上传：

```bash
cd game-design-methodology
pip install playwright beautifulsoup4 lxml
playwright install chromium

# 方式 1：导出后无头爬
python3 scripts/save_woa_login.py
python3 scripts/crawl_woa_knowledge.py --auth-file woa_auth.json --headless

# 方式 2：有界面浏览器边登边爬
python3 scripts/crawl_woa_knowledge.py
```

---

## 常见问题

| 问题 | 处理 |
|------|------|
| 云环境看不到浏览器 | 必须用方案 A，不能依赖云端弹窗 |
| `仍被 iOA 拦截` | `woa_auth.json` 过期，重新运行 `save_woa_login.py` |
| Cookie 多久过期 | 取决于 iOA 策略，失效后重新导出即可 |
| 爬取中断 | 自动续爬，进度在 `raw-articles-woa/crawl_state.json` |

---

## 文件说明

| 文件 | 作用 |
|------|------|
| `scripts/save_woa_login.py` | 本机登录并导出 `woa_auth.json` |
| `scripts/crawl_woa_knowledge.py` | 爬取主程序 |
| `woa_auth.json` | 登录状态（敏感，勿提交 Git） |
| `raw-articles-woa/` | 爬取结果 |
