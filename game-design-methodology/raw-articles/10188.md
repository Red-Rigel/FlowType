# Make the Impossible Possible- Create a New IP, Cook it AAA

> 来源: https://gameinstitute.qq.com/course/detail/10188
> 作者: Clark Yang 腾讯互娱 NExT工作室 创意总监及制作人

---

以下为分享实录：
我的题目就叫“Make The Impossible Possible”，在座的各位应该或多或少都有听过一个近年来讨论度比较高的问题，就是中国人能不能做AAA？我对这个问题的答案就在标题之中，在国内很多游戏从业者在过去由于产品和市场的定位，虽然具备个人实力，但是没有机会参与更高端更大型的项目。而对于整个行业而言，AAA游戏代表着什么，我们为什么要做3A？这个问题的答案不仅止于利益，在接下来的时间里我也就个人的一些思考来跟大家分享一下它的价值所在。
我在加入NExT工作室之前，曾先后在上海育碧，蒙特利尔育碧以及华纳公司参与过多个项目的研发，而这段在国外的工作经历也让我觉得自己不擅长做手游，所以直到NExT的出现，我才选择回国，因为NExT让我看到了一个在自己熟悉的领域开拓不同市场的可能性。
关于NExT工作室，自2017年至今已经上线了很多小而美的游戏，如《死神来了》，《尼山萨满》等等，每款游戏都有独立的风格特色，这也是NExT的立命根本，做让人眼前一亮的创新游戏。同时，在研发这些游戏的过程中，团队也逐步积累起了经验，所以从2018年3月份我们就启动了一款相对较大规模的游戏立项，它的定位就是AA，一个整个团队准备好在下一步挑战AAA的试炼石。关于这款游戏，我们先通过一个CG预告来了解一下。
在做SYNCED: Off-Planet（《重生边缘》）这款游戏时，一方面，团队希望通过尝试大规模的项目，为未来迎接3A的挑战做准备；另一外面，也希望能够将它从一个项目延伸至一件可以持续做下去的事，那就是IP。
一、     Create a New IP
当团队的目标是创造一个新的IP，一个大众都没有见过的原创IP时应该怎么做？这个Key Art图里就展示了团队对这个IP的定义。首先，名字是什么？改了近四十个版本后，我们敲定了SYNCED，这个名字需要让玩家能够抓住这款产品的核心概念。
在Key Art图中，游戏名的上面有一行小字“A NEXT INITIATIVE”，这句话所表达的是我们希望它不只是一款产品，而且可以持续下去，催生新的好内容。图的绘制也经过了很多次的迭代，找到静态与动态之间的平衡，比如小女孩所面临的绝境与其背后所存在的希望，这点后面也会进行扩展。
想要真正地打造一个全新的IP，太难了，无论是写小说，拍影视作品，还是研发3A游戏，都会谈到IP，但IP究竟怎么实现？
第一个图是Neil Druckmann（顽皮狗工作室的副总裁），他在一次采访当中回答过记者问的问题，那就是怎么做一个IP？他的回答是，做IP的方式是千万不要做IP，真的太难了。这点应该是所有有经验的开发者所持有的共识。既然如何，为什么还是要撞破头皮去做？因为IP的价值，远远大于产品本身。
产品可能在数月或者数年以后停止运营，但是IP是可持续的，这个大前提赋予了IP其价值。
接下来的内容讲IP创造的方法论，可能听起来不接地气，但我会尽量以通俗的说法来阐述。
做IP很重要的是内容，做IP就是在创造一个全新的世界，而这个世界中还需要有一些让观众意料不到的东西，这个世界为什么和观众看到过的世界不同？如果完全一模一样的世界，那为什么玩家不直接体验别的产品？
在这个前提之下，世界、角色就很重要，之所以专门把这张图（蝙蝠侠和小丑）拿过来，因为当我在华纳公司做《蝙蝠侠》的时候，跟他们团队经常讨论角色是谁？为什么这款产品中的角色与观众认知的角色不同？
因为这是冲突的来源，有了角色，英雄与反派之间的冲突，就一定会发生故事，
这个故事
存在于
这个
世界的体系之下，整个架构就立起来了。
这是创造IP最重要的一点，在全世界范围来讲，做得最好的就是大家都熟知的迪士尼。
IP
：《玩具总动员》
以《玩具总动员》为例，我们来看下世界、角色、故事怎么分析？
首先《玩具总动员》的世界是我们既熟悉又不熟悉的世界，我们都玩过玩具，电影呈现的场景是一个卧室，我们都曾经是小孩子，都玩过玩具；不熟悉的元素则是玩具会说话，玩具的生活是我们所不熟悉的另外一个世界，那么这个设定就创造了完完全全不同的世界，这是观众之前没有体验过的全新世界，而且这个世界中有自己的规则。
我们去分析一下的《玩具总动员》的规则是什么，这个世界的规则是什么？这个世界当中的规则是当真正的人类看到玩具的时候它们就不能动的，大家思考一下这个规则背后的逻辑，是因为不能让人类看到我能动，不能让人类看到玩具是有生命的；另外一个新的规则就是玩具之间说话人是听不到的，你可以发现直到第四季，仍然没有打破这个规则。
IP
：《星球大战》
回顾一下《星球大战》推出第一部的时候，它也创造了一个全新的世界，这个世界既能够让观众想象它，因为美国1969年的时候已经登月了，探索太空已经开始了，所以普通的观众可以想象出太空是什么样的；但是又不熟悉，比如光剑这种黑科技武器元素，它们也共同构建起了这个世界。
如果大家分析一下其他的内容也可以找到同样这三个重要的点：世界、角色、故事。但要打造一个游戏IP，我们还需要具备第四个元素——独特体验。因为游戏是可互动的，玩家手里有键盘、鼠标、手柄可以控制这个世界角色、人物的行为。
游戏IP：
《使命召唤》系列的独特体验是具有真实，强代入感的战争体验，当它在世界，角色，与故事的基础之上将战争体验做到了极致，便成为了一个经典IP。
《刺客信条》系列的宣传语是让玩家“身临其境地参与历史事件”，无论坐标是罗马还是埃及，它为玩家创造了一个拟真的体验。 再来，《神秘海域》的独特体验是“游戏化的印第安纳琼斯。《侠盗猎车手》系列则赋予了玩家超高自由度的体验，实现在现实生活中所不能做的事情。
从这些经典的游戏IP可以发现，区别于观众只能被动接收信息的影视IP，在游戏中，玩家时时刻刻在跟游戏中的角色互动。
回到《重生边缘》的KEY ART，我们当时就在思考，怎么绘制一副图，让观者形成冲击力？画面中我们可以看到一些生化人走了过来，地上全是尸体，但是仔细一看就发现，这些生化人的设计区别于其他作品，是硅基和碳基的结合物，而这也点出了《重生边缘》的核心游戏体验——玩家可以对敌人进行精神控制。
生化题材的经典作品有很多，思考如何将生化人做得更加花里胡哨没什么意思，那么换个角度，对于玩家而言，除了打和跑这两种玩法选择之外，是否还有其他可能？于是我们想到了控制，以及其控制后玩家能够进行的行为进行延伸，并从这个想法挖掘出了KEY ART。图中可以看出从女孩脖子上的连线可以接引到生化人身上并对他们实施控制，我们所要暗示玩家的就是这些对她产生生命威胁的生化人也可以在下一秒成为她的工具，帮助她脱离绝境。
对于团队而言，找到使这款产品区别于其他生化题材游戏的独特体验经过了很多轮的讨论和反复琢磨，这句宣传语： “可对敌人精神控制的多角色生存之旅” 便支撑起了《重生边缘》IP的世界、角色、故事。
游戏IP需要同时具备这四个元素，其框架才能成立。那么，在此前提下，世界、角色、故事，怎么才能够抓住玩家，引起他们的兴趣？否则无论你的游戏内容多丰富，玩家都不会在第一时间选择下载。要找到玩家的兴趣点，我们要攻克的问题有三个：
X：最吸引人的钩子是什么？需要找到这个点，比如对僵尸精神控制，这就是一个钩子，要把玩家钩住。
Y：我为什么在乎？要设定一些情景角色，让玩家对这个设定感兴趣，会倾注情感。
S：什么最能让玩家感到意外？
X、Y、S是综合体，无论你的IP对世界，角色，故事及体验的设定是怎么样的，都需要回归这三点，如果开发者能够抓住这三个关键，离构建一个强IP便很近了。希望大家能够记住这三个问题，在构建IP时不停地进行这三个灵魂拷问，而当你能够自信地给出答案之时，你的IP就是一个强有力的IP了。
当然构建IP只是一个开始，它可能需要花费大量的时间，这中间可能会有需要推翻迭代的过程，设计游戏总是需要我们不断思考，从不同的角度出发，论证我们的世界、角色、体验，是否逻辑自恰。而当我们认为这个IP的逻辑可以自恰时，又迎来了一个新的挑战，那就是当我们朝着AAA努力的同时，我们先要解决“AAA是什么”的问题。
二、       Cook it AAA
AAA的概念在国内外的游戏圈都经常被提及，前些年我们对它的定义是：A lot of money、A lot of people、A lot of time。而这意味着做AAA本身就是风险特别大的事，那么，换到今天，对于国内的游戏开发者来说，我们又该如何定义AAA？
关于这个问题的答案，我的观点是：
构建属于我们自己强有力的IP。
我们看到现在游戏市场上吸引人的IP全部是AAA的，国内有一些中小型IP，但是目前真的没有一款能够非常强有力可以持续非常长时间同时可以吸引人的IP，国内的团队还相对比较薄弱。
达到一流品质。
现在的市场其实给开发者提供了非常好的平台，这是可以达到的，而且我相信国内的开发者，无论是美术还是程序，我们是有输出一流产品品质的能力。
为全球主流市场定制。
游戏行业的整个市场环境之下，你会发现无论开发者和玩家都在相互融合，国内的玩家的水平开始慢慢提升，他们的审美、口味会慢慢接近国际化的审美和品位。对开发者来讲，残酷的现实就是现在就必须开始组建小规模团队做AA游戏，因为如果现在不抓住时间，很有可能在未来的几年，我们就将面临同育碧这类国际大厂同台竞争的情况。听上去虽然像是个伪命题，因为国内的很多游戏公司是比不上国际大厂的，但那些大厂同样是一步步打下来的技术积累和项目经验，我们能做的就是在还有时间的时候，抓住全球化的潮流，为自己博得一席之地。
构建强有力的IP
团队在最开始立项时，就花了非常大的力气，用上百页的文字来描述这个世界，描述什么是X、什么是Y，什么是S？这些文档是用英文编辑的，因为项目的目标就是国际市场，所以在最开始就以这个思维在逐步雕琢。游戏故事的版本有过无数次推翻重建的过程，它不可能是个一蹴而就的事情。
期间我们也参考了很多经典的作品，比如《生化危机》系列，其主线就是一个人创造了一个科技成果，这个科技成果落入了反派的手里，便结出了恶果。而这个后果也影响了角色或者敌人的设计思路。
许多《重生边缘》中的设定都是由此延伸而来，像是角色手臂上的臂环，其背后都有逻辑自恰的背景故事。它体现的游戏的风格设定，是我们希望呈现给玩家的世界。我要说的是，从研发设计上，整个臂环出现的样子，不是纯UI的样子，更多的是消息推送，这个当时并没有完全做出来的，这是设计和思路上展示的臂环的设计。
在这个风格设定的过程中，包括激烈的战斗场景，每一幅原画，都是为了整个产品服务的。
达到一流的品质
在技术上，我们团队也正在与英伟达进行一些合作，这里可以跟大家分享一个小故事。在做这个Demo的时候，英伟达希望跟我们联合发布，在有很多AAA产品同台亮相的发布会上，《重生边缘》将作为唯一一款全新的IP登场，而这对我们团队来说是个非常好的机会。但是在Demo的制作过程当中我们碰到了许多问题，因为里面所运用到的技术不是百分之百的成熟，团队和英伟达的程序员所要攻克的是连Epic都没有搞定的事。
当时有人质疑为什么要做这件事，因为Ray Tracing这个技术就是未来，几年以后的主流。现在先一步做技术积累，为的不是即时利益，而是长期的战略。
与此同时，大家看到这样美轮美奂的场景，我们在立项之初想达到的，对我们来说不光是做一个IP，做AAA的产品，是如何带领团队，真正背后的逻辑是需要建立世界一流的工业化体系。
工业化体系并不是小作坊可以做的事，涉及到很多技术的问题，人员分工的问题，涉及到很多管理的问题，对于我们而言，在整个一年多的开发过程当中，便是整个工业化体系的摸索过程。它的回报可能不是明天，对团队来说，至少在目前国内来讲，我们团队是最有实力和能力做到那样水平的团队。每一个标准件有非常明确的数值，这就是整体工业化流程的解读。高效产出资源，高效完成场景的多样性，都需要团队有人员进行持续地研究。
团队的成员曾经感叹过这个项目太累了，无数的技术文档需要调整修改，但是这个过程是必须的，没有这些文档的规范，团队就无法持续推动这个项目的前进。
为全球主流市场定制
《重生边缘》这款产品的研发过程，不仅是对团队能力的提升，同时我们也在于国外的媒体和朋友们进行交流，下面就是我们所取得的一个成果。
KOTAKU是一家很严格的媒体，当时他们的记者在最开始半个小时的试玩过程中面无表情，然后过了半个小时后，就开始从一些非常刁钻的角度开始提问，而当他完成试玩和提问之后，才真正地笑了。而之后他也在自己的专栏撰写了一篇文章来总结他的体验。
文章的结尾有一段话，大意是西方的游戏公司没有意识到腾讯游戏有多么强的产能，而且整个文章也十分走心，将NExT工作室的正确写法进行了编辑，这也让我们团队看到交流所带来的正向反馈，也给了我们鼓了一把劲。
未来团队要走的路很长，也很难走，但能够跨出这一步，挑战不一定成功的事情，第一次走到世界的舞台上，跟那些国际大厂的作品同台竞争，与此同时与一步步地在海外的游戏市场留下中国游戏的印象，这些积累会在未来一步步加速变化，终有一天，真正给全世界打造属于腾讯，属于中国人自己的强有力的IP。
